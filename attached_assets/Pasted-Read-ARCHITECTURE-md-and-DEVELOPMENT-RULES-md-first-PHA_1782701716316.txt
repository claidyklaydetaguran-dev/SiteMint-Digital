5. Responsiveness requirements

Desktop ≥ 1280px:

* Icon rail + expanded sidebar visible.
* Main content gets correct left offset.
* No horizontal overflow.
* Topbar actions visible.

Tablet 768px–1279px:

* Icon rail stays visible.
* Expanded sidebar can collapse.
* Clicking icon group opens the expanded sidebar drawer/panel.
* Search remains usable.
* Main content uses full available width.

Mobile < 768px:

* Hide permanent sidebar.
* Show mobile topbar with menu button.
* Menu button opens full-height slide-out nav.
* Nav closes after clicking a route.
* Quick actions should collapse into compact icons or overflow menu.
* No content should be clipped horizontally.
* Tables/cards should remain scrollable or stack gracefully.

General:

* Sidebar itself scrolls if nav items exceed height.
* Main page scroll remains independent.
* Active nav state works on all breakpoints.
* Coming Soon items are visibly disabled and not clickable.
* No nav overflow at 1024px, 1280px, 1440px.
