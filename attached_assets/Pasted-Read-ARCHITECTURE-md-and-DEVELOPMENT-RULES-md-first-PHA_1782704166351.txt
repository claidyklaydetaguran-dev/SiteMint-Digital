* /admin/crm/campaigns
* /admin/crm/campaign-builder
* /admin/crm/campaign-queue

1. /admin/crm/campaigns
   Shows Campaign History/List only:

* campaign list
* filters/sort
* open campaign
* analytics
* delete draft
* link to queue
* link to builder

2. /admin/crm/campaign-builder
   Shows builder/create mode:

* existing campaign builder UI

* save draft

If existing CrmCampaigns.tsx already contains the builder view, extract or route into it safely without duplicating logic.

3. /admin/crm/campaign-queue
   Shows queue UI:

* global queue by default
* status filters
* scheduled/queued/sent/failed/canceled/skipped
* send now
* edit queued message
* cancel message

Use existing CrmCampaignQueue component if possible.
