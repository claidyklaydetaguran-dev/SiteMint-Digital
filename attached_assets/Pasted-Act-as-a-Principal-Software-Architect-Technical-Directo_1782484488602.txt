TASK 2 — DATABASE SCHEMA

Create campaign persistence tables with Drizzle migration only.

Table: crm_campaigns

Fields:

* id
* name
* subject
* body
* status default "draft"
* created_at default now()
* updated_at default now()

Allowed statuses:

* draft
* ready
* archived

Table: crm_campaign_recipients

Fields:

* id
* campaign_id references crm_campaigns(id) on delete cascade
* lead_id references crm_leads(id) on delete cascade
* status default "selected"
* disc_style_used
* personalized_subject
* personalized_body
* created_at default now()
* sent_at nullable

Allowed recipient statuses:

* selected
* test_previewed
* sent
* skipped
* failed

Important:
No scheduling fields yet.
No automation fields yet.
No bulk send execution yet.
