PROJECT OVERVIEW
Build a standalone sales funnel (landing page + checkout + product delivery) for SiteMint Digital, an agency that builds AI-powered websites and business systems for service businesses. This funnel sells a new low-ticket digital product: an AI Prompt Toolkit for small and medium business owners — a done-for-you pack of Claude/ChatGPT prompts that help SMB owners handle marketing, sales, customer communication, and admin work without hiring extra staff.

This is modeled on the "prompt pack" funnel style used by creators like The Lead Suite (Claude for Closers, The Listing Machine, 30 Days of Content in a Day) — but positioned for general small business owners, not just real estate agents.

Goal: a single, high-converting page that takes a visitor from "I'm overwhelmed running my business" to "I just bought and received the toolkit" in one sitting, priced at $27 (adjust between $17–$47 as needed), paid via Stripe Checkout, with instant digital delivery.

TECH STACK
Frontend: Next.js (App Router) + Tailwind CSS — single-page funnel, mobile-first
Backend: Next.js API routes (or a small Express server if simpler in Replit)
Payments: Stripe Checkout (hosted, not custom card fields — fastest to ship and PCI-safe)
Delivery: Stripe webhook → on checkout.session.completed, trigger an email with a secure download link (use Resend or SendGrid) and log the sale
Storage: Start with a simple SQLite/Replit DB table for purchases (email, product, stripe session id, timestamp, download token). No need for a full user-auth system.
Env vars: STRIPE_SECRET_KEY, STRIPE_PUBLISHABLE_KEY, STRIPE_WEBHOOK_SECRET, RESEND_API_KEY (or SENDGRID_API_KEY), PRODUCT_PRICE_ID — pull these from Replit Secrets, never hardcode.
Since Stripe is already set up, only wire in the existing keys — don't scaffold new Stripe account creation flows.
